# Adaptive Cruise Control Using Fuzzy Logic

## Overview

This project presents the design and implementation of an Adaptive Cruise Control (ACC) system using Fuzzy Logic. The controller automatically adjusts the vehicle speed according to the distance from the vehicle ahead while maintaining a safe following distance and ensuring passenger comfort.

## Objectives

- Design an intelligent cruise control system.
- Maintain a safe distance from the leading vehicle.
- Automatically accelerate and decelerate.
- Improve driving comfort and safety.
- Implement fuzzy logic-based decision making.

## Features

- Automatic Speed Control
- Distance Monitoring
- Fuzzy Logic Controller
- Smooth Acceleration

- Collision Avoidance
- Intelligent Vehicle Control

- MATLAB

## Working Principle

The Adaptive Cruise Control system continuously monitors the distance between the host vehicle and the vehicle ahead. Based on vehicle speed and relative distance, the fuzzy logic controller determines whether the vehicle should accelerate, maintain speed, or decelerate. The controller processes the input variables through predefined fuzzy rules and generates an appropriate speed command, ensuring safe and smooth driving.

## Inputs

- Traffic condition

## Output

- Updated Vehicle Speed
